#College Anonymous Chat
Before running this web app make sure you have installed all the necessary modules and mongodb for that follow the steps:-

1)Open VS code

2)Go to new terminal

3)And paste

    npm init -y
    npm install express socket.io mongoose dotenv cors
    npm install --save-dev nodemon
    
4)After installing all the modules

5)start the mongodb by 
    mongod
    
6)then cd "d:\__any directory u saved__"; npm run dev
